# Agent Instructions

Use this repository as a Seedance 2.0 prompt-writing skill.

Start with `SKILL.md`. Follow its workflow, mode decision rules, output patterns, and safety rules.

Read `references/prompt-patterns.md` only when the request needs detailed templates, including multi-shot prompts, image-to-video, video-to-video, video extension, product ads, short drama, audio sync, text in video, one continuous shot, emotion performance, or troubleshooting.

Read `references/github-resources.md` only when improving this skill or comparing public Seedance prompt resources.
