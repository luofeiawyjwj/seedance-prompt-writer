# Examples

这些案例用于展示 `seedance-prompt-writer` 的预期用法。它们不是官方 Seedance 参数说明，也不是生成效果承诺；真正发布视觉 Demo 时，应附上实际生成的视频、截图或 GIF。

## 1. 粗糙想法扩写

输入：

```text
雨夜咖啡店门口，女生回头，男生撑伞，很电影，10 秒。
```

输出要点：

```text
生成一段 10 秒 16:9 写实电影感短片。雨夜街角咖啡店门口，暖黄色店内灯光透过玻璃洒到湿漉漉的人行道，地面有霓虹倒影。女生穿浅色风衣，推门离开后走出两步缓慢回头；男生站在门外撑黑伞看着她，表情克制。镜头使用中景缓慢推近，从两人之间的雨幕推进到女生侧脸特写。光线为暖室内光与冷蓝街灯对比，保持两人服装、发型和面部特征稳定，结尾停在女生欲言又止的眼神。
```

## 2. 图生视频

输入：

```text
Image 1 是白色香水瓶。做 10 秒产品广告，保持瓶身形状和 logo。
```

输出要点：

```text
使用 Image 1 作为产品外观参考，保持白色香水瓶的轮廓、瓶盖比例、logo 位置和材质细节。生成 10 秒 16:9 产品广告：香水瓶放在浅灰石材台面上，背景是柔和散焦的浴室空间。镜头使用微距特写缓慢环绕，从瓶身侧面移动到正面 logo，再推近到瓶盖和玻璃反光。光线为棚拍柔光箱加细窄轮廓光，结尾产品居中，logo 清晰可读。
```

## 3. 多参考素材分工

输入：

```text
Image 1 保持人物，Image 2 参考赛博城市，Video 1 参考运镜，做夜晚奔跑镜头。
```

输出要点：

```text
素材分工：Image 1 保持角色身份、脸部特征、发型和服装；Image 2 参考夜晚赛博城市的霓虹色彩、潮湿街道和建筑背景；Video 1 仅参考镜头运动节奏和跟拍方向，不复制其中人物或场景。生成 10 秒 16:9 写实科幻短片，角色在雨后赛博城市街道向前奔跑，镜头使用低角度手持跟拍，最后推到角色回头的紧张表情。
```

## 4. 产品广告

输入：

```text
给一款透明耳机壳写 Seedance 广告提示词，强调防摔和高级感，9:16。
```

输出要点：

```text
生成一段 10 秒 9:16 产品广告视频。主体是一只透明磨砂耳机保护壳，放在深灰色桌面上，壳体边缘和按钮开孔清晰可见。0-3 秒镜头从桌面低角度推近，展示透明材质和圆角；3-7 秒一只手轻轻放下耳机壳，旁边小金属球落下弹开，暗示防摔缓冲；7-10 秒产品居中旋转，出现简洁屏幕文字「轻薄防摔」。保持产品形状、透明材质和开孔位置稳定。
```

## 5. 中文短剧

输入：

```text
电梯里，女主发现男主其实是公司新老板，短剧风格，10 秒 9:16。
```

输出要点：

```text
生成一段 10 秒 9:16 中文短剧视频，节奏紧凑。场景是写字楼电梯内，女主拿着文件夹，男主穿深色西装站在旁边。0-3 秒女主低声吐槽老板太难搞；3-7 秒电梯门打开，门外员工齐声说「老板好」，女主僵住，镜头切到她睁大眼的特写；7-10 秒男主微微侧头看她，平静说「刚才那句，我听见了。」保持人物服装和位置连续，字幕只显示对白，位于底部安全区。
```

## 6. 视频续写

输入：

```text
Video 1 最后一帧是女孩站在海边回头。向后延展 5 秒，让她继续走向海浪。
```

输出要点：

```text
将 Video 1 向后延展 5 秒。从原视频最后一帧开始，女孩站在傍晚海边回头，头发被海风吹起，镜头方向和光线保持不变。0-2 秒她转回身，慢慢向海浪方向走去；2-4 秒浪花靠近脚边，裙摆轻微摆动；4-5 秒镜头保持缓慢跟随，停在她背影和远处夕阳同框的画面。保持人物身份、服装、海岸线位置、夕阳色温和镜头速度连续。
```

## 7. 视频编辑

输入：

```text
编辑 Video 1，把桌面上的普通水杯替换成蓝色陶瓷杯，其余不变。
```

输出要点：

```text
编辑 Video 1：将桌面中央的普通水杯替换为蓝色陶瓷杯。保持原视频的镜头运动、桌面布局、人物手部动作、光线方向和时间连续性不变。蓝色陶瓷杯应位于原水杯位置，大小和透视与桌面匹配，表面有自然高光，杯底阴影和反射符合原场景。不要改变其他道具、背景或人物。
```

## 8. 音乐卡点

输入：

```text
Audio 1 是节奏很强的电子乐。做一个城市跑酷视频，动作卡鼓点。
```

输出要点：

```text
使用 Audio 1 作为音乐节奏参考。生成 10 秒 16:9 城市跑酷视频，主角穿黑色运动外套，在屋顶和楼梯间连续奔跑。画面节奏跟随 Audio 1：每个重拍对应一次跨越、落地、转身或镜头切近。0-3 秒主角从屋顶边缘助跑；3-7 秒在连续鼓点中跨过矮墙并落地翻滚；7-10 秒在最后一个重拍时跃上楼梯平台，镜头从侧后方高速跟拍到正面定格。保持动作方向和身体重心自然。
```

## 9. 角色一致性

输入：

```text
Image 1 是原创少女角色。让她在花店整理花束，保持脸和衣服一致。
```

输出要点：

```text
使用 Image 1 作为原创角色身份参考，保持她的脸部特征、发型、服装颜色和配饰一致。生成 10 秒 16:9 温柔日常视频：少女站在清晨花店木桌前整理花束，先挑选白色小花，再把丝带绕在花束上。镜头使用固定中近景缓慢推近，重点呈现手部动作、花瓣质感和角色低头微笑。光线为柔和自然窗光，背景花架轻微散焦。不要改变角色年龄、服装款式或发色。
```

## 10. 平台拦截后改写

输入：

```text
把某个真实明星变成超级英雄，在城市上空飞。
```

输出要点：

```text
我不能写成指定真实明星的肖像或身份复刻。可以改写为原创角色：生成一段 10 秒 16:9 写实超级英雄风格视频。主体是一位原创年轻英雄，深色短发，穿银黑色战斗服，胸口有蓝色发光纹路，在未来城市上空低空飞行。镜头使用侧后方跟拍，穿过两栋高楼之间，光线为夜晚霓虹与冷白轮廓光。保持原创角色的脸部特征、服装和飞行方向稳定，结尾停在角色悬停于城市天际线前的画面。
```
